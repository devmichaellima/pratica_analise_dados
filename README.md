# Formação Prática em Análise de Dados

## Sobre o Repositório

Este repositório documenta uma jornada prática de desenvolvimento profissional voltada para formação acelerada em Análise de Dados.

O objetivo do treinamento é desenvolver competências técnicas e analíticas necessárias para atuar profissionalmente como Analista de Dados Júnior, utilizando desafios diários baseados em situações reais de empresa.

Todo o treinamento foi estruturado para simular problemas encontrados no mercado, com foco em:

- resolução de problemas de negócio
- análise de dados operacionais
- automação de processos
- criação de métricas
- manipulação de dados
- pensamento analítico
- organização de projetos
- construção de portfólio profissional

---

## Objetivo Principal

Desenvolver, em 30 dias, competências práticas para atuação profissional em dados através de desafios curtos, progressivos e conectados entre si.

O foco principal do treinamento é:

- preparação para vagas de Analista de Dados Júnior
- desenvolvimento de raciocínio analítico
- aprendizado baseado em projetos reais
- construção de experiência prática
- criação de portfólio técnico no GitHub

---

## Como o Treinamento Funciona

O treinamento é dividido em desafios diários.

Cada desafio:

- simula uma situação real de empresa
- possui contexto de negócio
- trabalha múltiplas competências ao mesmo tempo
- possui dificuldade progressiva
- é limitado para resolução rápida
- recebe code review técnico
- continua projetos anteriores

A proposta é desenvolver velocidade operacional, clareza técnica e capacidade de resolver problemas reais utilizando dados.
